# GuessTheFlag

## Project

### Description

Second project of [100 Days of SwiftUI](https://www.hackingwithswift.com/100/swiftui)

This project is a guessing game that helps users learn some of the many flags of the world.

## Development guidelines

### Scope

* Xcode: 12.5.1 (12E507)
* min iOS: 14.0+ 
* Swift: 5.0
